1. Introduction to Plotly

Plotly is a powerful Python library for creating interactive visualizations. Unlike static plots created with Matplotlib or Seaborn, Plotly charts allow users to interact with the data through features like hovering, zooming, panning, and filtering.

The key advantage of Plotly is that it generates web-based visualizations that can be embedded in web applications, dashboards, or Jupyter notebooks. This makes it ideal for data exploration and presentation where interactivity enhances understanding.

Interactive plots allow you to take your mouse over the graph and it shows you tooltips with all the data. This makes data exploration much more intuitive and user-friendly.

Why Use Plotly?

Feature Description Benefit
Interactivity Hover tooltips, zoom, pan, selection Better data exploration and understanding
Web-Based HTML/JavaScript rendering Easy embedding in websites and dashboards
Express API High-level interface for quick plots Faster development, less code
Graph Objects Low-level control for customization Fine-grained control over every element
Multiple Chart Types Line, bar, scatter, histogram, etc. Versatile visualization options

üìù Hinglish Summary

Plotly ek powerful library hai jo interactive graphs banane ke liye use hoti hai. Isme aap mouse hover karke data dekh sakte ho, zoom in-out kar sakte ho, aur bahut saare interactive features milte hain. Ye web-based hai toh aap isko easily websites aur dashboards mein use kar sakte ho. Plotly Express simple hai quick plots ke liye, jabki Graph Objects se aap detailed customization kar sakte ho.

üéØ Key Takeaways

  • Plotly creates interactive, web-based visualizations with hover tooltips, zooming, and panning capabilities
  • Plotly Express (px) provides a high-level, easy-to-use API for quick plotting
  • Graph Objects (go) offers low-level control for advanced customization
  • Interactive plots enhance data exploration and presentation
  • All Plotly charts are rendered as HTML/JavaScript, making them web-ready

üí° Practice Questions

Q1: What is the main advantage of Plotly over static plotting libraries like Matplotlib?
Plotly creates interactive visualizations with features like hover tooltips, zoom, pan, and selection, making data exploration more intuitive and user-friendly compared to static plots.
Q2: What are the two main APIs provided by Plotly?
Plotly Express (px) for high-level, quick plotting, and Graph Objects (go) for detailed customization and low-level control over visualization elements.
Q3: Why are Plotly visualizations called "web-based"?
Plotly renders charts as HTML/JavaScript, which makes them interactive in web browsers and easy to embed in websites, dashboards, and web applications.

2. Basic Interactive Plot using Plotly

2.1 Library Setup and Data Creation

To get started with Plotly, we need to import three essential libraries:

  • plotly.express - High-level plotting interface
  • pandas - Data manipulation and DataFrame creation
  • numpy - Numerical computations and array operations
import plotly.express as px
import pandas as pd
import numpy as np
We are using Plotly's express method for creating interactive plots. It's important to store your data in DataFrame format as it's quite efficient for handling multiple datasets.

Now let's create our data. We'll generate a sine wave for demonstration:

# Create data
x = np.linspace(0, 10, 100)  # 100 points from 0 to 10
y = np.sin(x)                 # Calculate sine values

# Store in DataFrame
data = pd.DataFrame({'x': x, 'y': y})

Here's what each line does:

  • np.linspace(0, 10, 100) - Creates 100 equally spaced values between 0 and 10
  • np.sin(x) - Calculates the sine value for each x point
  • pd.DataFrame() - Converts our data into a structured DataFrame with labeled columns
We have created 100 values between 0 and 10 for x-axis, and for each x value, we calculate the equivalent sine value. We capture both datasets into a single DataFrame structure, which is an inbuilt Python data structure.

2.2 Creating Your First Interactive Plot

Now we can create our first interactive line plot:

# Interactive line plot
fig = px.line(data, x='x', y='y', title='Interactive Sine Wave')
fig.show()

Let's break down the px.line() function parameters:

Parameter Value Purpose
data DataFrame object Source of data for the plot
x='x' Column name Specifies which column to use for x-axis
y='y' Column name Specifies which column to use for y-axis
title String Sets the title of the plot

The fig.show() method renders the plot in your browser or Jupyter notebook.

Interactive Features

Once the plot is displayed, you can interact with it in several ways:

  • Hover - Move your mouse over the line to see exact x and y values in a tooltip
  • Zoom In - Click and drag to select an area to zoom into
  • Zoom Out - Double-click anywhere to reset the zoom
  • Pan - Click the pan tool in the toolbar and drag to move around
  • Auto Scale - Automatically adjusts the axes to fit all data
  • Reset Axes - Returns to the original view
If you take the mouse over any particular portion of the graph, that portion will get highlighted. With double click, you can go back to the original view. Creating this highly interactive graph is not that difficult with Plotly's inbuilt library support.

üìù Hinglish Summary

Pehle humne teen libraries import ki - plotly.express, pandas aur numpy. Phir humne data create kiya using linspace function jo 0 se 10 tak 100 points generate karta hai. Iske baad sine values calculate ki aur sab kuch DataFrame mein store kiya. Finally, px.line() function use karke interactive plot bana diya jisme x-axis, y-axis aur title specify kiya. Plot show karne ke baad aap mouse hover, zoom in-out, aur pan kar sakte ho.

üéØ Key Takeaways

  • Import plotly.express, pandas, and numpy to get started
  • Use np.linspace() to create evenly spaced data points
  • Store data in pandas DataFrame for efficient handling
  • px.line() creates interactive line plots with minimal code
  • Interactive features include hover tooltips, zoom, pan, and auto-scale

3. Multiple Lines in One Plot

Often in data analysis, we need to compare multiple trends or datasets simultaneously. Plotly makes it easy to plot multiple lines on the same graph, allowing for direct visual comparison.

3.1 Creating Multiple Datasets

To plot multiple lines, we need to create a DataFrame with multiple y-columns:

x = np.linspace(0, 10, 100)
df = pd.DataFrame({
    'x': x,
    'sin(x)': np.sin(x),
    'cos(x)': np.cos(x)
})
We are creating two data sets - sin(x) and cos(x). Both functions share the same x-axis. If the range or domain is different between features, then obviously you cannot combine the data on the same plot.

In this DataFrame:

  • 'x' - Common x-axis values (0 to 10)
  • 'sin(x)' - First dataset (sine function values)
  • 'cos(x)' - Second dataset (cosine function values)

3.2 Visualizing Multiple Functions

Now we can plot both functions on the same graph:

fig = px.line(df, x='x', y=['sin(x)', 'cos(x)'], 
              title='Sine and Cosine Waves')
fig.show()

Notice that the y parameter now accepts a list of column names: ['sin(x)', 'cos(x)']. This tells Plotly to plot both columns as separate lines on the same graph.

We are using five columns out of the dataset - sepal length and sepal width for x and y coordinates, petal length for size of the point, species for color, and petal width for hover information.

3.3 Interactive Features

When you display the plot, you'll notice several powerful interactive features:

Feature How to Use Purpose
Color Coding Automatic assignment Each line gets a distinct color for easy identification
Legend Click on legend items Show/hide specific lines to focus on particular data
Hover Tooltips Move mouse over lines See exact values for all lines at that x-position
Zoom Click and drag Focus on specific regions of interest
Pan Use pan tool Navigate around the zoomed view
Reset Double-click Return to original view
You can zoom in by double clicking, zoom out, see the complete pan. You can select to show particular cosine or sine wave. If you zoom in, auto scale takes you out. You can reset the axis and hide particular labels or show them.

When to Use Multiple Lines

Multiple line plots are ideal for:

  • Trend Comparison - Comparing how different variables change over time
  • Mathematical Functions - Visualizing multiple functions simultaneously
  • Performance Metrics - Comparing KPIs or performance indicators
  • Scientific Data - Comparing experimental results or measurements

Aap legend se particular line hide ya show kar sakte ho, zoom kar sakte ho, aur hover se exact values dekh sakte ho.

üéØ Key Takeaways

  • Pass a list of column names to the y parameter: y=['col1', 'col2']
  • Plotly automatically assigns different colors to each line

4. Interactive Subplots using make_subplots

When you need to display multiple separate visualizations in an organized grid layout, Plotly's make_subplots function is the perfect tool. Unlike multiple lines on one plot, subplots allow each chart to have its own axes, scale, and data.

4.1 Creating Subplot Grids

First, we need to import the necessary modules:

from plotly.subplots import make_subplots
import plotly.graph_objects as go
import numpy as np

Now let's prepare data for four different mathematical functions:

# Data
x = np.linspace(0, 10, 100)
y1 = np.sin(x)           # Sine function
y2 = np.cos(x)           # Cosine function
y3 = np.exp(-x)          # Exponential decay
y4 = np.log(x + 1)       # Logarithmic function

Create a 2√ó2 subplot grid:

# Create subplot grid
fig = make_subplots(rows=2, cols=2, 
                    subplot_titles=('Sine', 'Cosine', 
                                   'Exponential Decay', 'Logarithm'))

Key parameters of make_subplots():

Parameter Purpose Example
rows Number of rows in the grid 2
cols Number of columns in the grid 2
subplot_titles Tuple of titles for each subplot ('Title1', 'Title2', ...)

4.2 Adding Multiple Traces

Now we add each plot to specific positions in the grid:

# Add plots to specific positions
fig.add_trace(go.Scatter(x=x, y=y1, name='sin(x)'), row=1, col=1)
fig.add_trace(go.Scatter(x=x, y=y2, name='cos(x)'), row=1, col=2)
fig.add_trace(go.Scatter(x=x, y=y3, name='exp(-x)'), row=2, col=1)
fig.add_trace(go.Scatter(x=x, y=y4, name='log(x+1)'), row=2, col=2)

Understanding add_trace():

  • go.Scatter() - Creates a scatter/line trace using graph objects
  • x, y - Data for the plot
  • name - Legend label for the trace
  • row, col - Position in the subplot grid (1-indexed)

Finally, customize the overall layout and display:

# Update layout
fig.update_layout(height=700, width=900, 
                 title_text="Interactive Subplots with Plotly")
fig.show()

Subplot Position Grid

[Grid Layout Visualization]
Row 1, Col 1: Sine | Row 1, Col 2: Cosine
Row 2, Col 1: Exponential Decay | Row 2, Col 2: Logarithm

üìù Hinglish Summary

Subplots mein hum multiple alag-alag charts ko ek organized grid mein dikha sakte hain. Pehle make_subplots() se grid banate hain jisme rows aur columns specify karte hain. Phir har chart ko add_trace() se add karte hain aur row, col parameters se position batate hain. Graph Objects (go) ka use karke Scatter plot banate hain. Har subplot ka apna axis aur scale hota hai. Last mein update_layout() se overall height, width aur title set karte hain.

üéØ Key Takeaways

  • Use make_subplots() to create organized grid layouts
  • Graph Objects (go) provide low-level control for subplot traces
  • add_trace() places each plot at specific row/col positions
  • Each subplot has independent axes and scales
  • update_layout() customizes overall figure properties

5. Mixing Chart Types in Subplots

One of Plotly's most powerful features is the ability to combine different chart types in the same subplot figure. This allows you to present diverse data visualizations side-by-side for comprehensive analysis.

5.1 Using Graph Objects

First, import the required libraries:

from plotly.subplots import make_subplots
import plotly.graph_objects as go
The subplot library allows us to combine multiple different types of charts together in one plot. Graph Objects (go) gives us the flexibility to use different chart types like line charts, bar charts, histograms, etc.

Graph Objects (go) provides access to all chart types:

  • go.Scatter() - Line and scatter plots
  • go.Bar() - Bar charts
  • go.Histogram() - Histogram plots
  • go.Pie() - Pie charts
  • go.Box() - Box plots
  • And many more...

5.2 Combining Line and Bar Charts

Let's create a figure with a line chart and a bar chart side-by-side:

# Create mixed subplots
fig = make_subplots(rows=1, cols=2, 
                    subplot_titles=('Line Chart', 'Bar Chart'))

# Add a line chart to first position
fig.add_trace(
    go.Scatter(x=[1, 2, 3, 4], 
               y=[10, 15, 13, 17], 
               mode='lines', 
               name='Line'),
    row=1, col=1
)

# Add a bar chart to second position
fig.add_trace(
    go.Bar(x=['A', 'B', 'C', 'D'], 
           y=[20, 14, 23, 25], 
           name='Bar'),
    row=1, col=2
)

# Update layout and display
fig.update_layout(title_text='Combining Chart Types', height=500)
fig.show()
We first create the subplot using make_subplots with one row and two columns. Then we add traces - first a line chart in row 1, column 1, and then a bar chart in row 1, column 2. Each chart type is treated as a trace that we add to the figure.

Understanding the Code Structure

Step Code Purpose
1. Create Grid make_subplots(rows=1, cols=2) Sets up 1√ó2 layout
2. Add Line Chart fig.add_trace(go.Scatter(...)) Adds line plot to position (1,1)
3. Add Bar Chart fig.add_trace(go.Bar(...)) Adds bar chart to position (1,2)
4. Customize fig.update_layout(...) Sets overall title and size
5. Display fig.show() Renders the figure

Interactive Features

When you display mixed chart types, each subplot maintains full interactivity:

  • Independent Zoom - Zoom in/out on each chart separately
  • Selective Legend - Hide/show specific chart traces
  • Individual Hover - Each chart type has appropriate hover information
  • Double-Click Reset - Reset each subplot independently
You can see there are two interactive charts combined together using subplots. You can zoom in, zoom out on each chart individually. You can hide bars or show them. Each chart maintains its own interactivity.

Common Chart Type Combinations

Combination Use Case Example
Line + Bar Trend vs. Category comparison Sales trend + Product categories
Scatter + Histogram Distribution with correlation Data points + Frequency distribution
Bar + Pie Absolute vs. Relative values Revenue by region + Market share
Line + Box Trend with statistical spread Average performance + Variability

üìù Hinglish Summary

Mixed chart types mein hum alag-alag tarah ke charts ko ek saath dikha sakte hain jaise line chart aur bar chart. Pehle make_subplots() se grid banate hain. Phir Graph Objects (go) ka use karke different chart types add karte hain - go.Scatter() line charts ke liye aur go.Bar() bar charts ke liye. Har chart ko add_trace() se specific position par place karte hain using row aur col parameters. Har subplot apni independent interactivity maintain karta hai jaise alag-alag zoom, hover, aur legend control.

üéØ Key Takeaways

  • Graph Objects enable mixing different chart types in subplots
  • Use go.Scatter() for line charts and go.Bar() for bar charts
  • Each chart type is added as a separate trace with add_trace()
  • Different chart types can have different data structures
  • Each subplot maintains independent interactivity

6. Interactive Scatter Plot (Hover, Zoom, Pan)

Scatter plots are excellent for visualizing relationships between variables and identifying patterns, clusters, and correlations. Plotly's interactive scatter plots allow encoding multiple dimensions of data through color, size, and hover information.

6.1 Working with Iris Dataset

The Iris dataset is a classic dataset in machine learning containing measurements of iris flowers. Let's load and explore it:

import plotly.express as px
import pandas as pd

# Load built-in Iris dataset

# Display the dataset
print(df.head())
The Iris dataset has 150 rows and 6 columns: sepal_length, sepal_width, petal_length, petal_width, species, and species_id. This gives us multiple attributes to work with for creating rich visualizations.

Dataset Structure

Column Name Data Type Description
sepal_length Float Length of sepal in cm
sepal_width Float Width of sepal in cm
petal_length Float Length of petal in cm
petal_width Float Width of petal in cm
species String Species name (setosa, versicolor, virginica)
species_id Integer Numeric species identifier

6.2 Multi-dimensional Data Visualization

Now let's create an interactive scatter plot that uses five different columns to visualize the data:

# Interactive scatter plot
fig = px.scatter(df, 
                x='sepal_width',           # X-axis
                y='sepal_length',          # Y-axis
                color='species',           # Color encoding
                size='petal_length',       # Size encoding
                hover_data=['petal_width'], # Additional hover info
                title='Interactive Iris Scatter Plot')

fig.show()

Understanding Multi-dimensional Encoding

This single plot encodes 5 dimensions of data:

Visual Element Data Column What It Shows
X-axis Position sepal_width Horizontal position of point
Y-axis Position sepal_length Vertical position of point
Color species Category/group membership
Size petal_length Magnitude through marker size
Hover Tooltip petal_width Additional information on hover
Out of six columns, we are using five to display information. Sepal length and sepal width define x and y coordinates, species defines color, petal length defines size of the point, and petal width shows as hover information. This is how you can use multiple columns to show a single data point with rich information.

Interactive Features

  • Color-coded Species - Each species automatically gets a distinct color
  • Size Variation - Larger points indicate larger petal length
  • Hover Information - Shows all encoded values plus additional petal_width data
  • Legend Selection - Click species names to show/hide groups
  • Zoom & Pan - Focus on specific clusters or outliers
  • Box/Lasso Select - Select multiple points for detailed inspection
You can zoom in and zoom out with double click. You can pick particular class to be shown based on species. Higher the petal length, bigger the size of the point. Smaller points have smaller petal length, bigger points have bigger petal length.

Benefits of Multi-dimensional Visualization

  1. Comprehensive View - See multiple relationships simultaneously
  2. Pattern Recognition - Identify clusters and groupings easily
  3. Outlier Detection - Unusual points stand out visually
  4. Category Comparison - Compare different groups at a glance
  5. Efficient Communication - Convey complex data in one visualization

X aur Y axes se position pata chalta hai, color se species identify hoti hai, size se petal length ka idea milta hai, aur hover karne par petal width bhi dikhta hai. Isse ek single plot mein bahut saari information efficiently convey ho jaati hai. Aap zoom, pan kar sakte ho, particular species ko hide/show kar sakte ho, aur patterns easily identify kar sakte ho.

üéØ Key Takeaways

  • px.scatter() creates interactive scatter plots with multi-dimensional encoding
  • Use color parameter to encode categorical data (species, groups)
  • Use size parameter to encode quantitative data (measurements)
  • hover_data allows adding extra information visible on hover
  • One scatter plot can effectively visualize 5+ dimensions of data
  • Interactive features like legend selection and zoom enhance exploration

7. Interactive Bar Chart with Hover Labels

Bar charts are ideal for comparing categorical data and showing quantities across different categories. Plotly's interactive bar charts add hover information, color coding, and customization options.

7.1 Creating Custom Data

Let's create our own dataset for a fruit inventory system:

import plotly.express as px
import pandas as pd

# Create custom data
data = {
    'Fruit': ['Apple', 'Banana', 'Orange', 'Grapes', 'Pineapple'],
    'Quantity': [120, 80, 150, 90, 60]
}

# Convert to DataFrame
df = pd.DataFrame(data)
For bar charts, we generally need categorical data. We are defining fruit names as categories and their quantities as numeric values. This data structure is perfect for bar chart visualization.

Our dataset structure:

Fruit (Category) Quantity (Value)
Apple 120
Banana 80
Orange 150
Grapes 90
Pineapple 60

7.2 Customizing Bar Chart Appearance

Now let's create a fully customized interactive bar chart:

# Create interactive bar chart
fig = px.bar(df, 
            x='Fruit',                  # Categories on x-axis
            y='Quantity',               # Values on y-axis
            text='Quantity',            # Display quantity on bars
            color='Fruit',              # Color by category
            title='Interactive Fruit Sales Chart')

# Customize text position and legend
fig.update_traces(textposition='outside')
fig.update_layout(showlegend=False)

fig.show()

Understanding the Parameters

Parameter Value Purpose
x 'Fruit' Categories displayed on x-axis
y 'Quantity' Height of bars
text 'Quantity' Text labels displayed on/near bars
color 'Fruit' Assigns different color to each category
title String Chart title

Advanced Customization Methods

fig.update_traces() - Modifies properties of chart traces:

fig.update_traces(textposition='outside')  # Places text above bars

Options for textposition:

  • 'outside' - Text appears above the bar
  • 'inside' - Text appears inside the bar at the top
  • 'auto' - Plotly automatically chooses best position

fig.update_layout() - Modifies overall figure properties:

fig.update_layout(showlegend=False)  # Hides the legend

Q2: What is the difference between update_traces() and update_layout()?
update_traces() modifies properties of the data traces (like bars, lines), while update_layout() modifies the overall figure properties (like title, legend, size, axes).
Q3: How do you display bar values inside the bars instead of outside?
Use fig.update_traces(textposition='inside') to place the text labels inside the bars at the top.
Q4: When would you set showlegend=True vs showlegend=False?
Set showlegend=True when you want users to toggle categories on/off. Set showlegend=False for cleaner presentation when legend is redundant (e.g., when x-axis already shows category names).

8. Interactive Line Chart with Range Slider

When working with time series data or large datasets, a range slider provides an intuitive way to focus on specific portions of the data while maintaining context of the overall trend.

8.1 Generating Time Series Data

First, let's import libraries and generate time series data:

import plotly.express as px
import numpy as np
import pandas as pd

# Generate Time Series Data
dates = pd.date_range("2024-01-01", periods=100)  # 100 days starting Jan 1, 2024
values = np.random.randn(100).cumsum()            # Random walk data

# Create DataFrame
df = pd.DataFrame({'Date': dates, 'Value': values})
For a slider with interactive data, it's good to have time series data. We use pd.date_range to generate dates and np.random.randn to generate random values. The cumsum() creates a cumulative sum which produces a random walk pattern.

Understanding the Data Generation

Function Purpose Output
pd.date_range() Generate date sequence 100 consecutive dates from start date
np.random.randn(100) Generate random normal values 100 random numbers from normal distribution
.cumsum() Cumulative sum Creates random walk pattern (trending data)

8.2 Implementing Range Slider

Now let's create the line chart with an interactive range slider:

# Create line chart
fig = px.line(df, x='Date', y='Value', 
             title='Time Series with Range Slider')

# Add range slider to x-axis
fig.update_xaxes(rangeslider_visible=True)

fig.show()

The key line is:

fig.update_xaxes(rangeslider_visible=True)

This adds an interactive range slider below the main plot on the x-axis.

The slider appears at the bottom on the x-axis. You can extend the right side or left side of this window, and that particular data will be shown. You can move this window and interact with the data more freely compared to just using zoom.

9. Interactive Histogram

A histogram displays the frequency distribution of numerical data by dividing the data range into bins and counting observations in each bin. Plotly's interactive histograms make it easy to explore data distributions.

9.1 Generating Random Data

Let's create a histogram using randomly generated data:

import plotly.express as px
import numpy as np
import pandas as pd

# Generate random data from normal distribution
data = np.random.randn(500)  # 500 random values

# Convert to DataFrame
df = pd.DataFrame({'Values': data})
We are generating 500 data points using normal random distribution. This will create data that follows a bell curve pattern, which is perfect for demonstrating histogram visualization.

The np.random.randn(500) function generates:

  • 500 random numbers
  • From a standard normal distribution (mean=0, std=1)
  • Values typically range from approximately -3 to +3
  • Follows a bell curve pattern

Creating the Histogram

Now let's create an interactive histogram:

# Create interactive histogram
fig = px.histogram(df, 
                  x='Values',                           # Data column
                  nbins=30,                             # Number of bins
                  color_discrete_sequence=['#636EFA'],  # Hex color
                  title='Interactive Histogram with Hover Tooltips')

fig.show()

Understanding Histogram Parameters

Parameter Value Purpose
x 'Values' Column containing numerical data to bin
nbins 30 Number of bins to divide the data range into
color_discrete_sequence ['#636EFA'] List of colors in hexadecimal format
title String Chart title

Number of Bins (nbins)

The nbins parameter controls histogram resolution:

Number of Bins Effect Best For
Too Few (5-10) Overly smooth, loses detail Very large datasets, general trends
Moderate (20-30) Balanced detail and clarity Most datasets (recommended)
Too Many (50+) Noisy, hard to interpret Large datasets needing fine detail

9.2 Color Customization with Hexadecimal

Hexadecimal color codes provide precise color control using RGB values:

color_discrete_sequence=['#636EFA']
You can define colors using hexadecimal code. For example, #636EFA where '63' is Red component, '6E' is Green component, and 'FA' is Blue component. Each component uses two hexadecimal digits (00-FF).

Understanding Hexadecimal Color Format

Format: #RRGGBB

Position Component Range Example
1-2 Red (R) 00 to FF (0-255) #FF0000 = Pure Red
3-4 Green (G) 00 to FF (0-255) #00FF00 = Pure Green
5-6 Blue (B) 00 to FF (0-255) #0000FF = Pure Blue

Example Color Modifications

# Pure blue (no red or green)
color_discrete_sequence=['#0000FF']

# Red + Green = Yellow
color_discrete_sequence=['#FFFF00']

# Remove all color components = Black
color_discrete_sequence=['#000000']

# Maximum all components = White
color_discrete_sequence=['#FFFFFF']
You can change the hexadecimal values to modify the color. For example, setting red and green to 00 leaves only blue. Setting red and green high creates yellow color. You can experiment with different combinations to get the color you want.

Interactive Histogram Features

  • Hover Tooltips - Shows exact count and bin range
  • Zoom - Focus on specific parts of the distribution
  • Automatic Binning - Plotly calculates bin ranges automatically
  • Distribution Shape - Visually identify normal, skewed, or uniform distributions
  • Frequency Analysis - See how many values fall in each range
The histogram automatically takes the count for each bin. Values are spreading from approximately minus 3 to plus 3, which is typical for normal distribution. It's looking like a bell curve, confirming our data follows normal distribution pattern.

Interactive histogram banane ke liye pehle random data generate karte hain using np.random.randn() jo normal distribution se 500 values deta hai. Phir isko DataFrame mein convert karte hain. px.histogram() function use karke histogram banate hain jisme 'nbins' parameter se kitne bins chahiye wo specify karte hain (30 ek balanced value hai). Color ko hexadecimal code se customize kar sakte hain - format hai #RRGGBB jaha RR red, GG green, aur BB blue component hai. Har component 00 to FF (0-255) tak ho sakta hai. Hover karne se exact count aur bin range dikhta hai. Normal distribution mein data -3 se +3 tak spread hota hai aur bell curve shape banta hai.

üéØ Key Takeaways

  • px.histogram() creates interactive frequency distribution plots
  • nbins parameter controls the number of bins (20-30 is typically optimal)
  • Hexadecimal colors format: #RRGGBB (00-FF for each component)
  • Plotly automatically calculates bin ranges and frequencies
  • Hover tooltips show exact counts and bin boundaries
  • Histograms reveal distribution shape (normal, skewed, uniform, etc.)
