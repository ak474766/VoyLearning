Introduction to Linear Algebra

Linear algebra is a branch of mathematics that forms the foundation of data science and AI applications. While it may sound abstract, it's essentially the mathematics of data, with matrices and vectors serving as the language of data.

Linear algebra focuses primarily on:

  • Linear combinations
  • The study of lines and planes
  • Vector spaces
  • Mappings required for linear transformations

History of Linear Algebra

Linear algebra is a relatively young field of mathematics that was initially formalized in the 1800s to find unknowns in systems of linear equations. The development of linear algebra can be traced through several key contributors:

  • Leibniz (1693) - Introduced the concept of determinants
  • Gabriel Cramer (1750) - Used determinants for solving linear systems, now known as Cramer's Rule
  • Carl Friedrich Gauss - Further developed the theory of solving linear systems with Gaussian elimination

The study of matrix algebra first emerged in England in the mid-1800s. Surprisingly, linear algebra only appeared in American graduate textbooks in the 1940s and in undergraduate textbooks in the 1950s, making it a relatively recent addition to mainstream mathematics education.

Summary

Linear algebra ek relatively young field hai jo 1800s mein formalize hua tha. Isme Leibniz ne determinants introduce kiye, Cramer ne linear equations ko solve karne ke liye determinants ka use kiya, aur Gauss ne Gaussian elimination method develop kiya. Matrix algebra England mein mid-1800s mein develop hui. Surprisingly, America mein ye sirf 1940s mein graduate textbooks aur 1950s mein undergraduate textbooks mein appear hui.

Understanding Vectors

Vector Notation and Formal Definition

In linear algebra, a vector is an ordered list of numbers. While in abstract linear algebra, vectors may contain other mathematical objects including functions, we will focus only on vectors comprising numbers for practical applications.

A vector can be represented in several ways:

As a column:    As a row:     Alternative notations:
   ⎡1.5⎤        [1.5 3.6 7.2 -9.1]       v = (1.5, 3.6, 7.2, -9.1)
   ⎢3.6⎥                                 v = [1.5, 3.6, 7.2, -9.1]
   ⎢7.2⎥
   ⎣-9.1⎦

Key terminology:

  • The numbers in the list are called elements, entries, coefficients, or components
  • The number of elements is the size, dimension, or length of the vector
  • A vector of size n is called an n-vector
  • Individual elements can be referenced by position (e.g., "the third entry is 7.2")

Summary

Linear algebra mein vector ek ordered list of numbers hota hai. Ise column form ya row form mein represent kar sakte hain. Vector mein jo numbers hote hain unhe elements, entries, coefficients ya components kehte hain. Vector mein jitne elements hote hain use vector ki size, dimension ya length kehte hain. Agar vector mein n elements hain to use n-vector kehte hain.

Zero, Ones, and Unit Vectors

There are several special types of vectors that are commonly used in linear algebra:

Zero Vector

A vector with all entries equal to 0 is called the zero vector, denoted by 0:

   ⎡0⎤        
   ⎢0⎥        or    [0 0 0]
   ⎣0⎦

Ones Vector

An n-vector with all entries equal to 1 is denoted by 1:

   ⎡1⎤        
   ⎢1⎥        or    [1 1 1]
   ⎣1⎦

Unit Vectors

A unit vector has only one entry equal to 1 and all others equal to 0. Unit vectors are denoted by ei where i is the position of the 1:

e₁ = ⎡1⎤     e₂ = ⎡0⎤     e₃ = ⎡0⎤
     ⎢0⎥          ⎢1⎥          ⎢0⎥
     ⎣0⎦          ⎣0⎦          ⎣1⎦

Professor mentioned in class: Unit vectors are particularly important for further operations on matrices, and we will see them again in later lectures when discussing linear transformations.

Key Takeaways

  • Vectors are ordered lists of numbers that form the basic building blocks in linear algebra
  • Vectors can be represented in multiple ways (column, row, or list notation)
  • Special vectors like the zero vector, ones vector, and unit vectors have important roles
  • Unit vectors are especially useful for operations and transformations

Summary

Linear algebra mein kuch special vectors hote hain: zero vector jisme sabhi entries zero hoti hain, ones vector jisme sabhi entries 1 hoti hain, aur unit vectors jinme sirf ek position par 1 hota hai aur baki sab jagah 0 hote hain. Unit vectors ko e₁, e₂, e₃, etc. ke roop mein denote karte hain jahan subscript 1 ki position batata hai. Ye special vectors aage ke matrix operations mein bahut useful honge.

Vectors in Python

Using NumPy Arrays

In Python, vectors can be represented using several data types, but NumPy arrays are best suited for linear algebra operations. Here are the main ways to create vectors in Python:

import numpy as np

# As a simple list (not ideal for linear algebra operations)
asList = [1, 2, 3]

# As a NumPy array without specific orientation
asArray = np.array([1, 2, 3])

# As a row vector (2D array with 1 row)
rowVec = np.array([[1, 2, 3]])

# As a column vector (2D array with 1 column)
colVec = np.array([[1], [2], [3]])

The asArray variable is an orientationless array, meaning it is neither a row nor a column vector but simply a 1D list of numbers in NumPy. While a simple Python list may seem like the easiest way to represent a vector, many linear algebra operations won't work on standard Python lists.

Professor mentioned in class: It's recommended to use NumPy arrays over Python lists for vector operations because NumPy provides better functionality for vector arithmetic, which Python lists do not support directly.

Summary

Python mein vectors ko represent karne ke liye NumPy arrays ka use karna sabse better option hai. Simple Python lists bhi use kar sakte hain, lekin unpe linear algebra operations perform nahi kar sakte. NumPy mein vectors ko different ways mein create kar sakte hain - orientationless array, row vector, ya column vector. NumPy import karke np.array() function se vectors create kiye jate hain.

Vector Orientations

In Python, it's important to understand the dimensionality and shape of vectors:

  • In mathematical terms, the dimensionality of a vector is the number of elements it contains
  • In Python, however, dimensionality refers to the number of geometric dimensions used to print out the numerical object
  • All vectors shown earlier are considered "two-dimensional arrays" in Python, regardless of the number of elements they contain
  • A list of numbers without orientation is considered a 1D array in Python
  • The mathematical dimensionality (number of elements) is called the length or shape of the vector in Python

We can explore these orientations by examining the shape of variables:

print(asList)                # Output: [1, 2, 3]
print(asArray)               # Output: [1 2 3]
print(rowVec)                # Output: [[1 2 3]]
print(colVec)                # Output: [[1]
                             #          [2]
                             #          [3]]

print(len(asArray))          # Output: 3
print(asArray.shape)         # Output: (3,)
print(rowVec.shape)          # Output: (1, 3)
print(colVec.shape)          # Output: (3, 1)

The shape of the variables gives you M rows and N columns in the format (M, N). For a 1D array, it simply shows the length as a tuple with one element.

Geometric Interpretation of Vectors

2-vectors (x₁, x₂) can represent a location or a displacement in 2D space. For example, if we have a vector (x₁, x₂), we can plot it on a 2D coordinate system where x₁ is along the x-axis and x₂ is along the y-axis.

Key Takeaways

  • NumPy is the preferred library for vector operations in Python
  • Understanding the difference between mathematical dimensionality and Python's concept of shape is crucial
  • Row vectors have shape (1, n) while column vectors have shape (n, 1)
  • Vectors can be visualized geometrically in 2D or 3D space

Summary

Python mein vectors ke orientation ko samajhna important hai. Mathematical terms mein vector ki dimensionality uske elements ki count hoti hai, lekin Python mein dimensionality ka matlab geometric dimensions hota hai. Python mein orientationless array ki shape (n,) hoti hai, row vector ki shape (1, n) hoti hai, aur column vector ki shape (n, 1) hoti hai. 2D vectors ko coordinate system par plot karke geometrically visualize kar sakte hain.

Vector Operations

Vector operations are essential for working with data in linear algebra. As the professor mentioned, "Vectors are like nouns; they are the characters in our linear algebra story. The fun in linear algebra comes from the verbs—the actions that breathe life into the characters. Those actions are called operations."

Vector Addition

Vector addition follows these properties:

  1. Commutative property: A + B = B + A
  2. Associative property: A + (B + C) = (A + B) + C
  3. Zero vector property: A + 0 = 0 + A = A
  4. Negative vector property: A + (-A) = 0

In Python, we can add vectors using NumPy:

import numpy as np

# Create vectors
A = np.array([1, 2, 3])
B = np.array([4, 5, 6])

# Vector addition
C = A + B  # Results in [5, 7, 9]

It's important to note that to add vectors, they must have the same dimensions (same number of elements).

Solution: [1+2, 3+4, 5+6] = [3, 7, 11]

Practice Question 2

Question: Can you add a row vector to a column vector as in the following example?

[1, 2, 3] + [[4],
              [5],
              [6]]

Solution: In standard mathematical linear algebra, you cannot directly add a row vector to a column vector because their dimensions don't match. However, in NumPy, broadcasting rules might allow this operation, resulting in a matrix where each element of the row vector is added to each element of the column vector.

Summary

Vector addition mein kuch important properties hain jaise commutative property (A + B = B + A), associative property (A + (B + C) = (A + B) + C), aur zero vector property (A + 0 = A). Vectors ko add karne ke liye unke dimensions same hone chahiye. Python mein NumPy ka use karke vectors ko easily add kar sakte hain. Generally, row vector ko column vector ke saath directly add nahi kar sakte, kyunki unke dimensions match nahi karte.

Scalar Multiplication and Addition

Scalar multiplication involves multiplying each element of a vector by a scalar value (a single number):

# Scalar multiplication
k = 2
kA = k * A  # Results in [2, 4, 6]

Some important properties of scalar multiplication include:

  • k(A + B) = kA + kB (distributive property)
  • (k + m)A = kA + mA (distributive property)
  • k(mA) = (km)A (associative property)
  • 1A = A (identity property)

While scalar addition is not formally defined in linear algebra (since vectors and scalars are different mathematical objects), numerical processing programs like Python will allow adding scalars to vectors. The operation adds the scalar to each element of the vector:

# Scalar addition (not formally defined in linear algebra but supported in Python)
A = np.array([1, 2, 3])
k = 5
result = A + k  # Results in [6, 7, 8]

Professor mentioned in class: While we can perform scalar addition in Python, it's important to understand that this is not a formal linear algebra operation but a convenience provided by programming languages.

Summary

Scalar multiplication mein hum ek vector ke har element ko ek scalar (single number) se multiply karte hain. Isme kuch properties hoti hain jaise distributive property aur associative property. Scalar addition linear algebra mein formally defined nahi hai, lekin Python jaise programming languages isme support karte hain, jahan scalar ko vector ke har element mein add kiya jata hai.

Vector Magnitude

The magnitude of a vector (also called its length or norm) represents the "size" of the vector. In geometric terms, it's the distance from the origin to the point represented by the vector.

For a vector v = [v₁, v₂, ..., vₙ], the magnitude is calculated as:

||v|| = √(v₁² + v₂² + ... + vₙ²)

This is also known as the Euclidean norm or the L2 norm.

For example, with the vector v = [3, 4], the magnitude is:

||v|| = √(3² + 4²) = √(9 + 16) = √25 = 5

In Python, we can calculate vector magnitude using NumPy:

import numpy as np

v = np.array([3, 4])
magnitude = np.linalg.norm(v)  # Results in 5.0

Scalar Multiplication and Vector Magnitude

When we multiply a vector by a scalar, the magnitude of the resulting vector follows this rule:

||kv|| = |k| × ||v||

For example, if v = [3, 4] and k = 3:

||3v|| = ||[9, 12]|| = √(9² + 12²) = √(81 + 144) = √225 = 15

Which equals |3| × ||v|| = 3 × 5 = 15

The absolute value of the scalar is used to ensure the magnitude remains positive even when multiplying by negative scalars.

Unit Vectors in Geometry

In some applications, we need a vector with a geometric length of one, called a unit vector. To convert any non-zero vector v into a unit vector û, we divide the vector by its magnitude:

û = v / ||v||

This process is called normalization.

Practice Question

Question: Calculate the magnitude of vector v = [1, 2, 2] and then find the unit vector in the same direction.

Solution:

||v|| = √(1² + 2² + 2²) = √(1 + 4 + 4) = √9 = 3

Unit vector û = v / ||v|| = [1/3, 2/3, 2/3]

Key Takeaways

  • Vector magnitude represents the "size" or "length" of a vector
  • The magnitude is calculated using the Euclidean norm (square root of sum of squares)
  • When a vector is multiplied by scalar k, its magnitude is multiplied by |k|
  • A unit vector has a magnitude of 1 and maintains the same direction as the original vector
  • In Python, the np.linalg.norm() function calculates vector magnitude

Summary

Vector magnitude (ya length/norm) vector ka "size" represent karta hai. Ise Euclidean norm se calculate kiya jata hai - vector ke har element ka square karke, sum karke, uska square root nikala jata hai. Jab hum vector ko ek scalar k se multiply karte hain, to magnitude |k| guna ho jata hai. Unit vector ek special vector hota hai jiska magnitude exactly 1 hota hai, aur direction original vector jaisa hi rehta hai. Python mein np.linalg.norm() function se magnitude calculate kiya ja sakta hai.

Practice Questions and Solutions

Quiz 1

(a) Given the vectors [2, 4, 6] and [1, 3, 5], what should be the result of the vector addition?

Solution: Adding corresponding elements: [2+1, 4+3, 6+5] = [3, 7, 11]

(b) Can you write down a Python code using NumPy to add these vectors?

import numpy as np

vector1 = np.array([2, 4, 6])
vector2 = np.array([1, 3, 5])
result = vector1 + vector2
print(result)  # Output: [3 7 11]

Quiz 2

Can you add a row vector to a column vector as in the following example?

[1, 2, 3] + [[4],
              [5],
              [6]]

Solution: In standard mathematical linear algebra, you cannot directly add a row vector to a column vector because they have different shapes. The row vector has shape (1, 3) while the column vector has shape (3, 1).

In NumPy, however, this operation might work due to broadcasting rules, resulting in a 3×3 matrix where each element of the row vector is added to each element of the column vector:

import numpy as np

row_vector = np.array([1, 2, 3])
column_vector = np.array([[4], [5], [6]])

# This will use broadcasting
result = row_vector + column_vector
print(result)
# Output:
# [[5 6 7]
#  [6 7 8]
#  [7 8 9]]

Additional Practice Question

Calculate the magnitude of the vector [3, 4, 0] and then find what scalar k would make the magnitude equal to 1.

Solution:

||v|| = √(3² + 4² + 0²) = √(9 + 16 + 0) = √25 = 5

For ||kv|| = 1, we need: |k| × ||v|| = 1

Therefore: |k| × 5 = 1

So k = 1/5 = 0.2

Key Takeaways from Practice Questions

  • Vector addition requires vectors of the same dimensionality
  • NumPy provides convenient methods for vector operations
  • Row and column vectors have different shapes and generally cannot be directly added
  • To normalize a vector (make its magnitude 1), divide it by its current magnitude
