1. Swapping Two Variables

Swapping is a fundamental operation where we exchange the values of two variables. This operation is frequently used in sorting algorithms and many other applications.

Problem: Given two variables A and B with values, swap their contents so that A holds B's value and B holds A's value.

Method 1: Using a Temporary Variable

This is the most intuitive approach. We use a third variable to temporarily store one value while we perform the swap.

Algorithm:

  1. Create a temporary variable (temp) to store the value of A
  2. Assign the value of B to A
  3. Assign the value stored in temp to B
def swap(a, b):
    print("Before swapping: A =", a, "and B =", b)
    
    # Using temporary variable
    temp = a
    a = b
    b = temp
    
    print("After swapping: A =", a, "and B =", b)
    return a, b

# Example: swap(5, 6)
# Output: A = 6, B = 5

Method 2: Without Using a Temporary Variable (Using Addition and Subtraction)

This method uses arithmetic operations to avoid using a third variable. It works for numeric values.

Algorithm:

  1. Add both variables and store in first variable (a = a + b)
  2. Subtract the second variable from the updated first variable to get the original value of the first variable (b = a - b)
  3. Subtract the updated second variable from the first variable to get the original value of the second variable (a = a - b)
def swap_without_temp(a, b):
    print("Before swapping: A =", a, "and B =", b)
    
    # Using addition and subtraction
    a = a + b
    b = a - b  # b now has the original value of a
    a = a - b  # a now has the original value of b
    
    print("After swapping: A =", a, "and B =", b)
    return a, b

# Example: swap_without_temp(100, 200)
# Output: A = 200, B = 100

Method 3: Python Specific Method

Python allows for elegant tuple assignment to swap variables directly.

def swap_python_way(a, b):
    print("Before swapping: A =", a, "and B =", b)
    
    # Python specific swap
    a, b = b, a
    
    print("After swapping: A =", a, "and B =", b)
    return a, b

# Example: swap_python_way(5, 6)
# Output: A = 6, B = 5

2. Counting Numbers in Collections

Counting is a fundamental operation in computing. Unlike humans who can visually count multiple objects at once, computers must iterate through collections one item at a time.

Problem: Count the number of elements in a collection that satisfy certain conditions.

General Algorithm for Counting:

  1. Initialize a counter variable to zero
  2. Iterate through each item in the collection
  3. For each item, check if it satisfies the condition
  4. If the condition is satisfied, increment the counter
  5. After the iteration completes, the counter holds the final count

Example: Counting Even Numbers

def count_even_numbers(numbers):
    count = 0
    for num in numbers:
        if num % 2 == 0:  # Check if number is even
            count = count + 1
    return count

# Example: count_even_numbers([15, 5, 3, 10, 16])
# Output: 2 (only 10 and 16 are even)

Example: Counting Numbers in a Range

def count_numbers_in_range(numbers, lower, upper):
    count = 0
    for num in numbers:
        if lower <= num < upper:  # Check if number is in the specified range
            count = count + 1
    return count

# Example: count_numbers_in_range([15, 5, 3, 10, 16], 5, 10)
# Output: 1 (only 5 is ≥ 5 and < 10)

Counting algorithms can be adapted to many different scenarios by modifying the condition checked for each element. This pattern is widely used in data analysis, statistics, and many other applications.

3. Working with Digits of a Number

Many algorithms require manipulating individual digits of a number. This includes summing digits, extracting specific digits, reversing numbers, and checking for palindromes.

3.1 Extracting Digits from a Number

Key Insight: We can extract the least significant digit (rightmost) using the modulo operation (% 10) and remove it using integer division (// 10).

Algorithm for Extracting Digits:

  1. To get the last digit: last_digit = number % 10
  2. To remove the last digit: number = number // 10 (integer division)
  3. Repeat until the number becomes 0

This approach works because modulo 10 gives the remainder when divided by 10, which is the last digit. Integer division by 10 effectively shifts the number right by one digit.

3.2 Summing the Digits of a Number

Problem: Given a number, find the sum of all its digits. For example, for 1234, the sum is 1+2+3+4 = 10.

def sum_of_digits(number):
    # Store the original number for later reference
    original_number = number
    
    # Initialize sum
    digit_sum = 0
    
    # Process until number becomes 0
    while number > 0:
        # Extract the last digit
        last_digit = number % 10
        
        # Add it to the sum
        digit_sum = digit_sum + last_digit
        
        # Remove the last digit
        number = number // 10
    
    print(f"Sum of digits in {original_number} is {digit_sum}")
    return digit_sum

# Example: sum_of_digits(1234)
# Output: Sum of digits in 1234 is 10

3.3 Reversing a Number

Problem: Given a number, reverse its digits. For example, 1234 becomes 4321.

Algorithm for Reversing a Number:

  1. Initialize a variable for the reversed number to 0
  2. Extract the last digit from the original number
  3. Add the extracted digit to the reversed number (after multiplying the current reversed number by 10)
  4. Remove the last digit from the original number
  5. Repeat until the original number becomes 0
def reverse_number(number):
    original_number = number
    reversed_num = 0
    
    while number > 0:
        # Extract the last digit
        last_digit = number % 10
        
        # Append it to the reversed number
        reversed_num = reversed_num * 10 + last_digit
        
        # Remove the last digit from original number
        number = number // 10
    
    print(f"Reverse of {original_number} is {reversed_num}")
    return reversed_num

# Example: reverse_number(1234)
# Output: Reverse of 1234 is 4321

The key step is to multiply the current reversed number by 10 before adding the new digit. This effectively shifts the existing digits left by one position.

3.4 Checking if a Number is a Palindrome

Problem: Determine if a number reads the same forward and backward. For example, 121 is a palindrome, but 123 is not.

def is_palindrome(number):
    original_number = number
    reversed_num = 0
    
    # First reverse the number
    while number > 0:
        last_digit = number % 10
        reversed_num = reversed_num * 10 + last_digit
        number = number // 10
    
    # Check if original equals reversed
    if original_number == reversed_num:
        print(f"{original_number} is a palindrome")
        return True
    else:
        print(f"{original_number} is not a palindrome")
        return False

# Example: is_palindrome(121)
# Output: 121 is a palindrome

# Example: is_palindrome(123)
# Output: 123 is not a palindrome

We can use our digit reversal algorithm and then simply compare the reversed number with the original. If they match, it's a palindrome.

4. Factorial Computation

Factorial is a mathematical operation denoted by n!, which is the product of all positive integers less than or equal to n.

Definition: n! = n × (n-1) × (n-2) × ... × 2 × 1

For example: 5! = 5 × 4 × 3 × 2 × 1 = 120

Algorithm for Computing Factorial:

  1. Initialize the result to 1 (not 0, as multiplication by 0 would always yield 0)
  2. Iterate from 1 to n (or n down to 1)
  3. For each number, multiply it with the current result
  4. Return the final product
def factorial(n):
    # Initialize result to 1 (not 0)
    fact = 1
    
    # Multiply numbers from 1 to n
    for i in range(1, n + 1):
        fact = fact * i
    
    print(f"Factorial of {n} is {fact}")
    return fact

# Example: factorial(5)
# Output: Factorial of 5 is 120

We could also implement it by counting down from n to 1:

def factorial_countdown(n):
    fact = 1
    
    # Multiply numbers from n down to 1
    for i in range(n, 0, -1):
        fact = fact * i
    
    print(f"Factorial of {n} is {fact}")
    return fact

# Example: factorial_countdown(5)
# Output: Factorial of 5 is 120

Applications: Factorials are used in combinatorics, probability, and many mathematical series like Taylor series expansions.

5. Fibonacci Series

The Fibonacci sequence is a series of numbers where each number is the sum of the two preceding ones, usually starting with 0 and 1.

Definition: The sequence begins: 0, 1, 1, 2, 3, 5, 8, 13, 21, ...

Each term F(n) = F(n-1) + F(n-2) for n > 1

Algorithm for Generating Fibonacci Series:

  1. Initialize the first two terms (0 and 1)
  2. To generate the next term, add the previous two terms
  3. Update the previous two terms for the next iteration
  4. Repeat until reaching the desired number of terms
def fibonacci_series(n):
    # Initialize the first two terms
    first = 0
    second = 1
    
    # Check if n is valid
    if n <= 0:
        print("Please enter a positive integer")
        return
    
    # Print the first term if n is at least 1
    if n >= 1:
        print(first, end=" ")
    
    # Print the second term if n is at least 2
    if n >= 2:
        print(second, end=" ")
    
    # Generate and print remaining terms
    count = 2  # We've already printed 2 terms
    while count < n:
        # Calculate the next term
        next_term = first + second
        print(next_term, end=" ")
        
        # Update for next iteration
        first = second
        second = next_term
        count += 1
    
    print()  # New line after printing all terms

# Example: fibonacci_series(8)
# Output: 0 1 1 2 3 5 8 13

Finding the Nth Fibonacci Number

Sometimes we only need the Nth number in the sequence, not the entire series.

def fibonacci_nth(n):
    # Handle special cases
    if n <= 0:
        print("Please enter a positive integer")
        return -1
    if n == 1:
        return 0
    if n == 2:
        return 1
    
    # Initialize the first two terms
    first = 0
    second = 1
    
    # Calculate the nth term
    count = 2
    while count < n:
        next_term = first + second
        first = second
        second = next_term
        count += 1
    
    print(f"The {n}th Fibonacci number is {second}")
    return second

# Example: fibonacci_nth(7)
# Output: The 7th Fibonacci number is 8

Historical Note: The Fibonacci sequence is related to the golden ratio and appears in many natural phenomena and artistic compositions. It has been studied for hundreds of years and continues to be of interest in mathematics and computer science.

Summary and Key Takeaways

These fundamental algorithms serve as building blocks for more complex problems and larger applications:

  • Swapping Variables: Critical for sorting and many other algorithms
  • Counting in Collections: Essential for data analysis and statistics
  • Digit Manipulation: Key for various number processing tasks
  • Factorial Computation: Used in combinatorics and mathematical series
  • Fibonacci Series: Has applications in nature, art, and computer science

For each of these problems, we've seen how to:

  1. Break down the problem into smaller steps
  2. Identify the critical operations needed
  3. Implement efficient algorithms
  4. Test with sample inputs to verify correctness
