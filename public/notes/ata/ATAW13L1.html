Introduction

Today's lecture covered the conversion between characters and their numerical representations in Python. This conversion is fundamental to many programming operations and applications.

Key Concepts:

  • Characters are stored as numbers internally using ASCII/Unicode encoding
  • Every character has a unique numerical value assigned to it by international standards
  • Converting between characters and numbers is essential for encryption, data processing, and numerical operations
  • Characters are one-byte entities in computing that can represent alphabets, numbers, or special symbols

Examples of Character-to-Number Mappings:

'A' → 65
'1' → 49
'@' → 64
Characters in Python are represented as strings, even single characters.
Python unifies the concept of characters and strings, treating them all as string objects.

The ord() Function

Python provides the ord() function which converts a character into its Unicode code point (a number).

Syntax and Examples:

# Get the Unicode code point of a character ord('A') # Output: 65 ord('1') # Output: 49 ord('z') # Output: 122

How It Works:

The ord() function accepts a single character and returns its ASCII/Unicode value.
This numerical value is what gets stored in binary form in the computer's memory.
The function name "ord" stands for "ordinal" value of the character.

Note:

ASCII (American Standard Code for Information Interchange) is an older standard that evolved into Unicode. Modern systems use Unicode which can represent characters from virtually all languages.

Converting Strings to Numbers

When working with longer strings, we can convert each character to its numerical value using list comprehension.

# Convert each character in "Hello" to its ASCII value text = "Hello" ascii_values = [ord(c) for c in text] print(ascii_values) # Output: [72, 101, 108, 108, 111]

Explanation:

The list comprehension iterates through each character in the string.
For each character, ord() returns its numerical value.
The result is a list of integers representing the ASCII/Unicode values.
This technique is useful for processing text data numerically.

Using Loops for Conversion

A more verbose but sometimes clearer approach is to use explicit loops for conversion.

# Iterate through characters and print their ASCII values text = "Python" for char in text: print(f"Character: {char}, ASCII: {ord(char)}")
Character: P, ASCII: 80 Character: y, ASCII: 121 Character: t, ASCII: 116 Character: h, ASCII: 104 Character: o, ASCII: 111 Character: n, ASCII: 110

When to Use Loops:

Loops offer more control and are easier to understand for beginners.
They allow for additional processing or conditions for each character.
Useful when you need to track position or perform different operations on different characters.

Summing ASCII Values of a String

We can compute the sum of all ASCII values in a string, which is useful for checksums and simple encryption techniques.

# Sum the ASCII values of all characters in "abc" text = "abc" ascii_sum = sum(ord(c) for c in text) print("Total ASCII Sum:", ascii_sum) # Output: Total ASCII Sum: 294

Calculation Breakdown:

ord('a') = 97
ord('b') = 98
ord('c') = 99
97 + 98 + 99 = 294

Applications:

Simple checksum algorithms for data validation
Basic encryption and hashing techniques
Text analysis and pattern recognition

Extracting Digits from a String

When working with mixed content strings, we often need to extract just the numeric portions.

# Extract only the digits from a mixed string text = "A1B2C3" numbers = [ord(c) - ord('0') for c in text if '0' <= c <= '9'] print(numbers) # Output: [1, 2, 3]

How It Works:

The conditional '0' <= c <= '9' filters out non-digit characters.
For digit characters, we calculate ord(c) - ord('0').
This formula converts the character digit to its actual integer value:

ord('1') - ord('0') = 49 - 48 = 1

ord('2') - ord('0') = 50 - 48 = 2

ord('3') - ord('0') = 51 - 48 = 3

The result is a list of actual integers extracted from the string.

Converting a Whole Number String to Integer

While Python provides int() for string-to-integer conversion, understanding the manual process is valuable for algorithmic thinking.

# Manually convert a string of digits to an integer def string_to_int(s): num = 0 for char in s: num = num * 10 + (ord(char) - ord('0')) return num print(string_to_int("1234")) # Output: 1234

Algorithm Explanation:

For input "1234", the algorithm works as follows:

  1. Initialize num = 0
  2. Process '1': num = 0*10 + (49-48) = 0 + 1 = 1
  3. Process '2': num = 1*10 + (50-48) = 10 + 2 = 12
  4. Process '3': num = 12*10 + (51-48) = 120 + 3 = 123
  5. Process '4': num = 123*10 + (52-48) = 1230 + 4 = 1234
This algorithm mimics how we manually convert digits to numbers: multiply by 10 and add the next digit.
It's similar to the C/C++ atoi() function (ASCII to Integer).

Handling Non-Numeric Characters in Conversion

In real-world applications, we need to handle mixed or invalid input when converting strings to numbers.

# Safely convert a string to an integer, handling non-numeric characters def safe_string_to_int(s): num = 0 for char in s: if '0' <= char <= '9': # Only process digits num = num * 10 + (ord(char) - ord('0')) else: return "Invalid Input" return num print(safe_string_to_int("567")) # Output: 567 print(safe_string_to_int("56a7")) # Output: Invalid Input

Improvements:

The function checks if each character is a digit before processing it.
If a non-digit character is found, it returns an error message instead of an incorrect result.
This approach prevents silent failures and unexpected behavior.
In production code, you might use exceptions instead of returning error strings.

C/C++ Equivalents: atoi() and Alternatives

The manual conversion methods we've explored are similar to functions in C/C++ standard libraries.

atoi() Function:

In C/C++, atoi() (ASCII to Integer) converts string representations to integers.
It works similarly to our manual conversion method, processing digits one by one.

Problems with atoi():

  • No error handling — "abc" returns 0 without any indication of failure
  • Cannot detect large numbers (integer overflow issues)
  • Only works with whole numbers

Better Alternatives:

strtol()

Handles errors & large numbers

Points to invalid characters

C++ stoi()

Modern C++ alternative

Has exception handling

Python's int() function is more robust than C's atoi() and handles errors with exceptions.

Summary and Key Takeaways

  • Character Representation: Characters are stored as numbers (ASCII/Unicode) internally in computers
  • ord() Function: Converts characters to their numerical values in Python
  • Digit Conversion: To convert digit characters to integers, subtract the ASCII value of '0'
  • Manual Methods: Understanding manual conversion provides insight into how computers process text
  • Error Handling: Always check for invalid input when processing characters as numbers
  • Applications: Character-to-number conversion is essential for encryption, data processing, and text manipulation
