Introduction to Number Systems

A number system is a writing system for expressing numbers; it is a mathematical notation for representing numbers of a given set, using digits or other symbols in a consistent manner.

Key Point: The base of a number system indicates the total number of symbols available in that system.

Common number systems include:

  • Decimal (Base 10): Uses digits 0-9
  • Binary (Base 2): Uses digits 0-1
  • Octal (Base 8): Uses digits 0-7
  • Hexadecimal (Base 16): Uses digits 0-9 and letters A-F

Prof mentioned: Binary is the "mother tongue" of computers, while decimal is our natural system!

Decimal System (Base 10)

The decimal system is the standard system for denoting integer and non-integer numbers. It is what we use in our everyday calculations.

  • Also called the base 10 system
  • Uses digits from 0 to 9
  • The positional value is based on powers of 10

Example: The decimal number 347

3 × 102 + 4 × 101 + 7 × 100

3 × 100 + 4 × 10 + 7 × 1 = 347

This is our natural number system - we've been using this since childhood!

Binary System (Base 2)

The binary system uses only two digits: 0 and 1. It is used by almost all modern computers and computer-based devices.

  • Also called the base 2 system
  • Uses only digits 0 and 1
  • The positional value is based on powers of 2
  • It's the "native language" of computer systems

Example: The binary number 1101

1 × 23 + 1 × 22 + 0 × 21 + 1 × 20

1 × 8 + 1 × 4 + 0 × 2 + 1 × 1 = 13 in decimal

Binary is crucial for computing! Everything in computers is ultimately represented in binary.

Octal System (Base 8)

The octal number system uses digits from 0 to 7. It is often used as a shorthand for binary numbers.

  • Also called the base 8 system
  • Uses digits from 0 to 7
  • The positional value is based on powers of 8
  • Each octal digit represents 3 binary digits (bits)

Example: The octal number 173

1 × 82 + 7 × 81 + 3 × 80

1 × 64 + 7 × 8 + 3 × 1 = 64 + 56 + 3 = 123 in decimal

Prof explained that octal gives us higher-level abstractions of binary (3 bits = 1 octal digit)!

Hexadecimal System (Base 16)

The hexadecimal system uses digits from 0 to 9 and letters A to F (representing values 10 to 15). It is widely used in computing.

  • Also called the base 16 system
  • Uses digits 0-9 and letters A-F (representing values 10-15)
  • The positional value is based on powers of 16
  • Each hexadecimal digit represents 4 binary digits (bits)

Example: The hexadecimal number 2F3

2 × 162 + F × 161 + 3 × 160

2 × 256 + 15 × 16 + 3 × 1 = 512 + 240 + 3 = 755 in decimal

Hexadecimal is great for compact representation of binary data (4 bits = 1 hex digit)!

Base Conversion Overview

Base conversion involves converting a number from one base to another. Understanding these conversions is essential for computer science and programming.

Common conversions:

  • Decimal to Binary
  • Binary to Decimal
  • Decimal to Octal
  • Octal to Decimal
  • Decimal to Hexadecimal
  • Hexadecimal to Decimal

Prof emphasized that interconversion skills are crucial for programming and computer science!

Decimal to Binary Conversion

To convert a decimal number to binary, you repeatedly divide the decimal number by 2 and record the remainder for each division. The remainders read from bottom to top form the binary number.

Example: Convert decimal 25 to binary

25 ÷ 2 = 12 remainder 1

12 ÷ 2 = 6 remainder 0

6 ÷ 2 = 3 remainder 0

3 ÷ 2 = 1 remainder 1

1 ÷ 2 = 0 remainder 1

Reading remainders from bottom to top: 11001

Therefore, 25 in decimal = 11001 in binary

# Python code for decimal to binary conversion def decimal_to_binary(n): binary = "" while n > 0: remainder = n % 2 binary = str(remainder) + binary # Prepend remainder to result n = n // 2 # Integer division return binary # Example usage print(decimal_to_binary(25)) # Output: 11001

Note: Python provides the built-in bin() function to convert decimal to binary. It adds the prefix '0b' to indicate binary representation.

bin(25) returns '0b11001'

bin(25)[2:] returns '11001' (removes the '0b' prefix)

Important: When implementing the conversion yourself, be sure to use integer division (// in Python) to ensure n eventually becomes 0!

Remember to read remainders from bottom to top (or reverse the string if accumulating from top to bottom)!

Binary to Decimal Conversion

To convert a binary number to decimal, sum the products of each binary digit and its corresponding power of 2.

Example: Convert binary 101010 to decimal

1 × 25 + 0 × 24 + 1 × 23 + 0 × 22 + 1 × 21 + 0 × 20

32 + 0 + 8 + 0 + 2 + 0 = 42

Therefore, 101010 in binary = 42 in decimal

# Python code for binary to decimal conversion def binary_to_decimal(binary): decimal = 0 for i in range(len(binary)): digit = int(binary[len(binary)-1-i]) # Get digit from right to left decimal += digit * (2 ** i) # Multiply by corresponding power of 2 return decimal # Example usage print(binary_to_decimal("101010")) # Output: 42

Note: Python provides the built-in int() function with a base parameter to convert from other bases to decimal.

int('101010', 2) returns 42

Decimal to Octal Conversion

To convert a decimal number to octal, repeatedly divide the decimal number by 8 and record the remainder for each division. The remainders read from bottom to top form the octal number.

Example: Convert decimal 136 to octal

136 ÷ 8 = 17 remainder 0

17 ÷ 8 = 2 remainder 1

2 ÷ 8 = 0 remainder 2

Reading remainders from bottom to top: 210

Therefore, 136 in decimal = 210 in octal

# Python code for decimal to octal conversion def decimal_to_octal(n): octal = "" while n > 0: remainder = n % 8 octal = str(remainder) + octal # Prepend remainder to result n = n // 8 # Integer division return octal # Example usage print(decimal_to_octal(136)) # Output: 210

Note: Python provides the built-in oct() function to convert decimal to octal. It adds the prefix '0o' to indicate octal representation.

oct(136) returns '0o210'

oct(136)[2:] returns '210' (removes the '0o' prefix)

Octal to Decimal Conversion

To convert an octal number to decimal, sum the products of each octal digit and its corresponding power of 8.

Example: Convert octal 345 to decimal

3 × 82 + 4 × 81 + 5 × 80

3 × 64 + 4 × 8 + 5 × 1 = 192 + 32 + 5 = 229

Therefore, 345 in octal = 229 in decimal

# Python code for octal to decimal conversion def octal_to_decimal(octal): decimal = 0 for i in range(len(octal)): digit = int(octal[len(octal)-1-i]) # Get digit from right to left decimal += digit * (8 ** i) # Multiply by corresponding power of 8 return decimal # Example usage print(octal_to_decimal("345")) # Output: 229

Note: Python provides the built-in int() function with a base parameter to convert from other bases to decimal.

int('345', 8) returns 229

Decimal to Hexadecimal Conversion

To convert a decimal number to hexadecimal, repeatedly divide the decimal number by 16 and record the remainder for each division. The remainders read from bottom to top form the hexadecimal number.

Example: Convert decimal 269 to hexadecimal

269 ÷ 16 = 16 remainder 13 (D)

16 ÷ 16 = 1 remainder 0

1 ÷ 16 = 0 remainder 1

Reading remainders from bottom to top: 10D

Therefore, 269 in decimal = 10D in hexadecimal

# Python code for decimal to hexadecimal conversion def decimal_to_hex(n): hex_chars = "0123456789ABCDEF" hex_result = "" while n > 0: remainder = n % 16 hex_result = hex_chars[remainder] + hex_result # Prepend remainder to result n = n // 16 # Integer division return hex_result # Example usage print(decimal_to_hex(269)) # Output: 10D

Note: Python provides the built-in hex() function to convert decimal to hexadecimal. It adds the prefix '0x' to indicate hexadecimal representation.

hex(269) returns '0x10d'

hex(269)[2:].upper() returns '10D' (removes the '0x' prefix and converts to uppercase)

Hexadecimal to Decimal Conversion

To convert a hexadecimal number to decimal, sum the products of each hexadecimal digit and its corresponding power of 16.

Example: Convert hexadecimal 1AB to decimal

1 × 162 + A(10) × 161 + B(11) × 160

1 × 256 + 10 × 16 + 11 × 1 = 256 + 160 + 11 = 427

Therefore, 1AB in hexadecimal = 427 in decimal

# Python code for hexadecimal to decimal conversion def hex_to_decimal(hex_str): hex_str = hex_str.upper() # Convert to uppercase for consistency hex_chars = "0123456789ABCDEF" decimal = 0 for i in range(len(hex_str)): digit = hex_chars.index(hex_str[len(hex_str)-1-i]) # Get value of hex digit decimal += digit * (16 ** i) # Multiply by corresponding power of 16 return decimal # Example usage print(hex_to_decimal("1AB")) # Output: 427

Note: Python provides the built-in int() function with a base parameter to convert from other bases to decimal.

int('1AB', 16) returns 427

Python Implementation - Built-in Functions

Python provides built-in functions for number system conversions. Here's a summary of the functions and their usage:

Conversion Function Example Result
Decimal to Binary bin() bin(25) '0b11001'
Decimal to Octal oct() oct(25) '0o31'
Decimal to Hexadecimal hex() hex(25) '0x19'
Binary to Decimal int(str, 2) int('11001', 2) 25
Octal to Decimal int(str, 8) int('31', 8) 25
Hexadecimal to Decimal int(str, 16) int('19', 16) 25

Prefix notations in Python:

  • Binary: 0b or 0B
  • Octal: 0o or 0O
  • Hexadecimal: 0x or 0X
# Python examples with prefixes a = 234 # Decimal number # Converting to other bases using built-in functions print(bin(a)) # Output: '0b11101010' print(oct(a)) # Output: '0o352' print(hex(a)) # Output: '0xea' # Removing prefixes print(bin(a)[2:]) # Output: '11101010' print(oct(a)[2:]) # Output: '352' print(hex(a)[2:]) # Output: 'ea' # Converting from other bases to decimal print(int('0b11101010', 2)) # Output: 234 print(int('0o352', 8)) # Output: 234 print(int('0xea', 16)) # Output: 234 # Direct conversion without prefix print(int('11101010', 2)) # Output: 234 print(int('352', 8)) # Output: 234 print(int('ea', 16)) # Output: 234

Prof highlighted that these built-in functions make conversion easy, but understanding the underlying process is crucial!

Custom Implementation - Decimal to Binary

Let's look at a step-by-step implementation of decimal to binary conversion without using built-in functions:

def decimal_to_binary(n): # Initialize an empty string to store binary representation b = "" # Repeat until n becomes 0 while n != 0: # Get remainder when n is divided by 2 r = n % 2 # Prepend remainder to the result b = str(r) + b # Integer division of n by 2 n = n // 2 return b # Test with sample numbers print(decimal_to_binary(17)) # Output: 10001 print(decimal_to_binary(25)) # Output: 11001 print(decimal_to_binary(234)) # Output: 11101010

Important implementation notes:

  1. Use integer division (//) to ensure n eventually becomes 0
  2. Prepend each remainder to the result string to get the correct order
  3. If you accumulate remainders in reverse order, you'll need to reverse the string at the end

Alternative approach (accumulating and then reversing):

def decimal_to_binary_alt(n): # Initialize an empty string to store binary representation b = "" # Repeat until n becomes 0 while n != 0: # Get remainder when n is divided by 2 r = n % 2 # Append remainder to the result (in reverse order) b = b + str(r) # Integer division of n by 2 n = n // 2 # Reverse the string to get correct binary representation return b[::-1] # Test print(decimal_to_binary_alt(17)) # Output: 10001

Prof mentioned that some numbers like 17 (10001) are palindromes in binary, so it might trick you into thinking your code is correct even when it's not! Always test with multiple examples!

Summary of Base Conversions

  • Decimal to Binary: Divide by 2, record remainders in reverse order
  • Binary to Decimal: Sum products of binary digits and powers of 2
  • Decimal to Octal: Divide by 8, record remainders in reverse order
  • Octal to Decimal: Sum products of octal digits and powers of 8
  • Decimal to Hexadecimal: Divide by 16, record remainders in reverse order
  • Hexadecimal to Decimal: Sum products of hexadecimal digits and powers of 16

Practical applications:

  • Binary is the fundamental language of computers
  • Octal and hexadecimal provide more compact representations of binary
  • Hexadecimal is commonly used in programming for representing memory addresses, colors, and binary data
  • Understanding number systems is crucial for computer science and programming

Important takeaway: Being able to convert between number systems is a fundamental skill in computer science! Practice these conversions to build your algorithmic thinking!
