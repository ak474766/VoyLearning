Introduction to Programming Concepts

Before diving into Python lists, let's review some fundamental programming concepts:

  • Algorithm: A set of rules or steps used to solve a problem
  • Data Structure: A particular way of organizing data in a computer

Data structures are crucial as they determine:

  • How data is arranged in memory
  • How data is accessed by the program
  • The overall efficiency of the program

Lists (or arrays in other languages) are one of the most fundamental data structures.

What is Not a "Collection"?

Most variables in programming contain a single value - a scalar value. When we assign a new value, the old one is overwritten:

>>> x = 2
>>> x = 4
>>> print(x)
4

When we assign a new value to a variable, the old value is replaced completely.

A List is a Kind of Collection

Unlike scalar variables, collections allow us to store multiple values in a single variable:

  • A collection allows us to put many values in a single "variable"
  • Collections are convenient for carrying multiple values in one package
friends = ['Joseph', 'Glenn', 'Sally']
carryon = ['socks', 'shirt', 'perfume']

Note: Lists in Python are more like linked lists in data structure terms, not strictly arrays. Lists are more flexible because they can grow and shrink, and can contain elements of different types.

List Constants

Lists are defined with specific syntax in Python:

  • List constants are surrounded by square brackets [ ]
  • Elements in the list are separated by commas
  • A list element can be any Python object - even another list
  • A list can be empty
>>> print([1, 24, 76])
[1, 24, 76]
>>> print(['red', 'yellow', 'blue'])
['red', 'yellow', 'blue']
>>> print(['red', 24, 98.6])
['red', 24, 98.6]
>>> print([1, [5, 6], 7])
[1, [5, 6], 7]
>>> print([])
[]

Key Insight: Unlike arrays in some other languages, Python lists can hold elements of different types (heterogeneous data).

for i in [5, 4, 3, 2, 1]:

Lists and Definite Loops - Best Pals

Lists work perfectly with definite loops (for loops) in Python:

friends = ['Joseph', 'Glenn', 'Sally']
for friend in friends:
    print('Happy New Year:', friend)
print('Done!')
Happy New Year: Joseph
Happy New Year: Glenn
Happy New Year: Sally
Done!

The loop variable (friend) takes on each value in the list one by one, making it easy to process list elements sequentially.

Looking Inside Lists

Just like strings, we can access any single element in a list using an index specified in square brackets:

>>> friends = ['Joseph', 'Glenn', 'Sally']
>>> print(friends[1])
Glenn

Important: List indexing starts at 0, not 1!

'Joseph' is at index 0, 'Glenn' is at index 1, 'Sally' is at index 2

Lists are Mutable

A key difference between strings and lists:

  • Strings are "immutable" - we cannot change the contents of a string; we must create a new string to make any change
  • Lists are "mutable" - we can change an element of a list using the index operator
# Strings are immutable
>>> fruit = 'Banana'
>>> fruit[0] = 'b'
Traceback TypeError: 'str' object does not support item assignment
>>> x = fruit.lower()
>>> print(x)
banana

# Lists are mutable
>>> lotto = [2, 14, 26, 41, 63]
>>> print(lotto)
[2, 14, 26, 41, 63]
>>> lotto[2] = 28
>>> print(lotto)
[2, 14, 28, 41, 63]

How Long is a List?

The len() function takes a list as a parameter and returns the number of elements in the list:

>>> greet = 'Hello Bob'  # Also works with strings
>>> print(len(greet))
9
>>> x = [1, 2, 'joe', 99]
>>> print(len(x))
4

The len() function tells us the number of elements of any set or sequence (such as a string, list, etc.).

Using the range Function

The range() function returns a list of numbers that range from zero to one less than the parameter:

>>> print(list(range(4)))
[0, 1, 2, 3]
>>> friends = ['Joseph', 'Glenn', 'Sally']
>>> print(len(friends))
3
>>> print(list(range(len(friends))))
[0, 1, 2]

A Tale of Two Loops...

There are two common ways to loop through a list:

# Method 1: Direct iteration (preferred when you just need the values)
friends = ['Joseph', 'Glenn', 'Sally']
for friend in friends:
    print('Happy New Year:', friend)

# Method 2: Using indices (useful when you need the position)
for i in range(len(friends)):
    friend = friends[i]
    print('Happy New Year:', friend)

Best Practice: Use direct iteration when you only need the values. Use index-based iteration when you need to know the position or modify elements.

Concatenating Lists Using +

We can create a new list by adding two existing lists together:

>>> a = [1, 2, 3]
>>> b = [4, 5, 6]
>>> c = a + b
>>> print(c)
[1, 2, 3, 4, 5, 6]
>>> print(a)  # Original list is unchanged
[1, 2, 3]

Lists Can Be Sliced Using :

Just like strings, lists can be sliced using the colon operator:

>>> t = [9, 41, 12, 3, 74, 15]
>>> t[1:3]  # Elements from index 1 up to (but not including) index 3
[41, 12]
>>> t[:4]   # Elements from beginning up to (but not including) index 4
[9, 41, 12, 3]
>>> t[3:]   # Elements from index 3 to the end
[3, 74, 15]
>>> t[:]    # All elements (creates a copy of the list)
[9, 41, 12, 3, 74, 15]

Remember: Just like in strings, the second number is "up to but not including"

You can also use negative indices for slicing:

  • t[-1] refers to the last element
  • t[-2] refers to the second-last element
  • t[-3:-1] gives elements from the third-last up to (but not including) the last element

Building a List from Scratch

We can create an empty list and then add elements using the append method:

>>> stuff = list()  # Creates an empty list
>>> stuff.append('book')
>>> stuff.append(99)
>>> print(stuff)
['book', 99]
>>> stuff.append('cookie')
>>> print(stuff)
['book', 99, 'cookie']

The list stays in order and new elements are added at the end of the list.

Is Something in a List?

Python provides two operators that let you check if an item is in a list:

>>> some = [1, 9, 21, 10, 16]
>>> 9 in some
True
>>> 15 in some
False
>>> 20 not in some
True

These are logical operators that return True or False and do not modify the list.

Lists are in Order

A list can hold many items and keeps those items in the order until we do something to change the order:

>>> friends = ['Joseph', 'Glenn', 'Sally']
>>> friends.sort()  # Sort the list alphabetically
>>> print(friends)
['Glenn', 'Joseph', 'Sally']
>>> print(friends[1])
Joseph

Note: The sort() method modifies the list in place - it doesn't return a new list. This is different from string methods, which return new strings.

Built-in Functions and Lists

There are a number of functions built into Python that take lists as parameters:

>>> nums = [3, 41, 12, 9, 74, 15]
>>> print(len(nums))  # Number of elements
6
>>> print(max(nums))  # Largest element
74
>>> print(min(nums))  # Smallest element
3
>>> print(sum(nums))  # Sum of all elements
154
>>> print(sum(nums)/len(nums))  # Average
25.6

Important: These functions only work with homogeneous lists of numeric types. They won't work if the list contains strings or other non-numeric types.

Example: Computing Average with Lists

Here's a comparison of two approaches to computing an average:

# Method 1: Traditional approach
total = 0
count = 0
while True:
    inp = input('Enter a number: ')
    if inp == 'done':
        break
    value = float(inp)
    total = total + value
    count = count + 1

average = total / count
print('Average:', average)

# Method 2: Using list
numlist = list()
while True:
    inp = input('Enter a number: ')
    if inp == 'done':
        break
    value = float(inp)
    numlist.append(value)

average = sum(numlist) / len(numlist)
print('Average:', average)

The second method is more concise and demonstrates the power of using list functions.

Best Friends: Strings and Lists

Strings and lists work well together, especially with the split() method:

>>> abc = 'With three words'
>>> stuff = abc.split()
>>> print(stuff)
['With', 'three', 'words']
>>> print(len(stuff))
3
>>> print(stuff[0])
With

Split breaks a string into parts and produces a list of strings. We can access a particular word or loop through all the words:

>>> for w in stuff:
...     print(w)
...
With
three
words

Splitting with Delimiters

>>> line = 'A lot of spaces'
>>> etc = line.split()
>>> print(etc)
['A', 'lot', 'of', 'spaces']

>>> line = 'first;second;third'
>>> thing = line.split()
>>> print(thing)
['first;second;third']
>>> print(len(thing))
1

>>> thing = line.split(';')
>>> print(thing)
['first', 'second', 'third']
>>> print(len(thing))
3
  • When you do not specify a delimiter, multiple spaces are treated like one delimiter
  • You can specify what delimiter character to use in the splitting

The Double Split Pattern

Sometimes we split a line one way, and then grab one of the pieces of the line and split that piece again:

# Example: Extracting domain from an email address
# From dipsankarb@iitj.ac.in Sat Jan 5 09:14:16 2008

words = line.split()
email = words[1]      # Extract the email address
pieces = email.split('@')  # Split the email address at '@'
print(pieces[1])     # Print the domain part

# Output: iitj.ac.in

This pattern is very useful for parsing complex strings like email headers, log files, etc.
