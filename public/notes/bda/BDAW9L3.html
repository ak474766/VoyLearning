Lecture Notes: Data Transformations

Data Transformations

  • After cleaning the data (dealing with duplicates, missing data, formatting, and fixing errors), we often have to make additional transformations to the data before starting the analysis.
  • These transformations are essential steps in the data preparation process and come in different forms.
  • Two major types of transformations:
    • Changing the scale of variables (for continuous variables)
    • Encoding categorical variables (for non-numeric data)

Hinglish Summary:

Data transformation ke baare mein aaj hum seekhenge. Cleaning ke baad, data ko analyze karne se pehle uska transformation karna zaroori hota hai. Do main tarah ke transformations hote hain - continuous variables ka scale change karna aur categorical variables ko numeric form mein convert karna. Ye steps data preparation ka important hissa hain.

Transforming Continuous Variables

Continuous variables often need transformation for the following reasons:

  • To make them more interpretable or comparable
  • To manage their distribution
  • To handle skewness of variables

Changing the Scale of Variables

We change the scale of variables when:

  • Variables have different units of measurement and need to be on a common scale
  • A few large numbers dominate the dataset

Standardization (Z-score Normalization)

Z = (Xi - X̄) / S

Where:

  • Xi is the original value in the dataset
  • X̄ (X-bar) is the sample mean of the dataset
  • S is the standard deviation of the dataset

This transformation brings all continuous variables to the same scale and makes them follow something similar to a normal distribution.

Min-Max Scaling

Xscaled = (Xi - Xmin) / (Xmax - Xmin)

Where:

  • Xi is the original value
  • Xmin is the minimum value in the dataset
  • Xmax is the maximum value in the dataset
  • (Xmax - Xmin) is the range of the dataset

Uses:

  • Rescales data to a fixed range, typically [0,1]
  • Used when working with fixed-interval data (e.g., percentages and proportions)
  • When preserving relative relationships between values is important

Caution: This method is sensitive to outliers, as extreme values determine the range

Logarithmic Transformations

Xtransformed = log(Xi)

When values include zeros:

Xtransformed = log(Xi + 1)

Uses:

  • Useful for handling highly skewed data
  • Compresses large values while expanding small values
  • Makes the distribution closer to normal, which is helpful for statistical analysis

Hinglish Summary:

Continuous variables ko transform karne ke teen main tarike hain. Standardization mein hum har value se mean subtract karke usko standard deviation se divide karte hain, jisse saare variables ek hi scale par aa jate hain. Min-Max scaling mein data ko 0 aur 1 ke beech mein convert kiya jata hai, lekin isme outliers ka dhyan rakhna padta hai. Logarithmic transformation skewed data ke liye useful hai, kyunki ye bade values ko compress aur chote values ko expand karta hai. In sabka use data analysis mein comparison aur interpretation ko better banane ke liye kiya jata hai.

Encoding Categorical Variables

Categorical variables need to be transformed into numerical representations for statistical analysis.

  • There are multiple ways to transform categorical variables into numerical representations
  • Essential for handling data and conducting the analysis correctly
  • Many statistical software packages cannot handle categorical variables directly
  • Encoding should be done in a way that preserves the underlying information in these categories

One-Hot Encoding (Dummy Variables)

Creates binary indicator variables for each category:

  • For a categorical variable with K categories, create K binary columns
  • Variable Xi takes value 1 if it belongs to category K, 0 otherwise
  • Used to encode variables with a small number of categories
  • Examples: Gender (male/female), Place of stay (Rural/Urban)

Example of One-Hot Encoding:

Original categorical variable: Gender (Male, Female)

After one-hot encoding:

Original is_Male is_Female
Male 1 0
Female 0 1

Note: In practice, we often only use K-1 binary columns to avoid multicollinearity.

Label Encoding

Assigns integer values to categories:

Category Red Blue Green
Encoded Value 1 2 3
  • Used when you have discrete variables without any logical ordering
  • The actual numbers assigned don't matter as long as you're consistent throughout the analysis

Ordinal Encoding

Maps categories to ordered integer values based on logical ranking:

Category Low Medium High
Encoded Value 1 2 3
  • Used when you have ordinal variables with meaningful order
  • Examples: Satisfaction ratings, education levels, size categories (S, M, L, XL)
  • The numbers assigned must follow the logical ranking of the categories

Hinglish Summary:

Categorical variables ko numerical form mein convert karna zaroori hai kyunki statistical software aise data ko seedhe process nahi kar sakte. Iske teen main tarike hain: One-Hot Encoding mein har category ke liye 0 ya 1 wala separate column banta hai, jaise gender ke liye is_Male aur is_Female. Label Encoding mein har category ko ek number assign kiya jata hai, jaise Red=1, Blue=2, Green=3. Ordinal Encoding mein categories ko unke natural order ke hisaab se number diye jate hain, jaise Low=1, Medium=2, High=3. Sahi encoding chunne ke liye categories ke beech relationship samajhna zaroori hai.

Recap

In this lecture, we covered:

  • Data transformations as a critical step after data cleaning
  • Transformations for continuous variables:
    • Standardization (Z-score normalization)
    • Min-Max scaling
    • Logarithmic transformations
  • Encoding methods for categorical variables:
    • One-Hot Encoding (Dummy Variables)
    • Label Encoding
    • Ordinal Encoding

Aaj ki class mein humne data transformation ke baare mein seekha. Data cleaning ke baad ye ek important step hai. Continuous variables ke liye Standardization, Min-Max scaling aur Logarithmic transformation seekhe. Categorical variables ke liye One-Hot Encoding, Label Encoding aur Ordinal Encoding ke baare mein jaana. Ye sare concepts data analysis mein bahut useful hain kyunki inse data ko sahi format mein convert kiya ja sakta hai taki statistical analysis accurate ho sake.
