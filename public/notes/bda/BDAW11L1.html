Data Analytics Basics: Arranging & Exploring Data

Lecture 4.9 Notes

Introduction to Data Arrangement

In previous lectures, we covered data gathering and cleaning. This lecture focuses on the steps needed after you have your dataset ready:

  • Arranging data in formats suitable for analysis
  • Visual inspection of datasets (part of exploratory data analysis)
  • Identifying structures and patterns
  • Grouping data into bins to observe record behavior

These tasks can be performed using various tools:

  • Excel (demonstrated in this lecture)
  • Programming languages like R or Python
  • AI tools (to be covered in lab sessions)

Toh dosto, aaj hum baat karenge data ko arrange karne ke baare mein. Aapne data collect kar liya hai, clean kar liya hai, ab kya? Ab aapko apne data ko aise format mein rakhna hai jisse aap uska analysis kar sake. Visual inspection, patterns dekhna, aur grouping karna - ye sab exploratory data analysis ka hissa hai. Aur ye kaam Excel, Python, R ya AI tools se ho sakta hai!

Sample Dataset Overview

For this lecture, we'll use a small sample dataset containing transaction records:

Purchase Date Transaction ID Product ID Product Name Amount
2/2/2025 53701 382 Watermelon 75
2/1/2025 53666 226 Grapes 29
2/2/2025 53701 984 Potato 22
2/1/2025 53666 453 Banana 14
... ... ... ... ...

Important observations about this dataset:

  • Purchase dates are not in chronological order
  • Transaction IDs are repeated (one transaction can contain multiple products)
  • Multiple transactions can occur on the same date
  • Each row represents one product purchased within a transaction

This resembles typical supermarket transaction data where:

  • Multiple people make purchases (transactions) on a given day
  • One person can purchase multiple products in a single transaction

Humne ek sample dataset liya hai jisme transaction data hai. Is data mein date, transaction ID, product ID, product name aur amount hai. Ek baar data ko dekhne se hum samajh sakte hain ki ek din mein kai transactions hote hain, aur ek transaction mein kai products ho sakte hain. Bilkul supermarket ke data jaisa! Abhi ye data kisi khaas order mein nahi hai, isliye hume ise arrange karna padega analysis ke liye.

Basic Data Arranging Techniques

1. Sorting Data

Sorting helps arrange data in a specific order based on the values in a column:

  • Sorting by Amount (Smallest to Largest):
    • Select the "Amount" column
    • Use Sort function (smallest to largest)
    • Expand selection to include all columns
  • Benefits:
    • Easily see minimum and maximum values
    • Quickly count occurrences (e.g., bananas purchased 4 times)
    • Identify value distribution patterns

Example of data sorted by Amount (smallest to largest):

Purchase Date Transaction ID Product ID Product Name Amount
2/1/2025 53666 453 Banana 14
2/2/2025 53701 453 Banana 14
2/1/2025 53668 453 Banana 14
2/2/2025 53703 453 Banana 14
... ... ... ... ...

You can also sort in descending order (largest to smallest) to see the highest values first.

Sorting yani data ko kisi ek column ke values ke hisaab se arrange karna. Jaise, amount ke hisaab se sort karne par hum dekh sakte hain ki sabse sasta product banana hai (₹14) aur sabse mehnga watermelon (₹75). Sorting se hume ye bhi pata chalta hai ki kitne baar kaun sa product bika - jaise banana 4 baar, potato 5 baar, grapes 4 baar, aur watermelon 3 baar. Large datasets mein sorting bahut useful hoti hai jab aap visually sab data ek saath nahi dekh sakte.

2. Filtering Data

Filtering allows you to display only records that meet specific criteria:

  • How to apply a filter:
    • Select all data
    • Use the Filter command (adds dropdown arrows to column headers)
    • Click the dropdown for the column you want to filter
    • Select/deselect values to show/hide
  • Example filtering scenarios:
    • Filter by Purchase Date (e.g., show only 2/1/2025 transactions)
    • Filter by Transaction ID (show products in one transaction)
    • Filter by Product Name (show only specific products)

Example of data filtered by Purchase Date (2/1/2025):

Purchase Date Transaction ID Product ID Product Name Amount
2/1/2025 53666 226 Grapes 29
2/1/2025 53666 453 Banana 14
2/1/2025 53667 984 Potato 22
2/1/2025 53668 382 Watermelon 75
... ... ... ... ...

Filtering matlab data me se specific cheezein dikhana. Jaise agar hum sirf 1 February ke transactions dekhna chahte hain, toh filter laga sakte hain. Filter lagane se hum dekh sakte hain ki us din kitne transactions hue, kaun kaun se products bike, aur kitne amount ke bike. Phir hum Transaction ID ke hisaab se bhi filter laga sakte hain aur dekh sakte hain ki ek particular transaction mein kitne products the. Ye hierarchical data analysis ka ek bahut important part hai!

3. Grouping Data at Different Hierarchical Levels

By combining filtering and sorting, you can analyze data at different hierarchical levels:

  • First Level: Filter by date to see all transactions on a specific day
  • Second Level: Within a filtered date, sort by Transaction ID to group products by transaction
  • Third Level: Look at products within each transaction

This approach allows you to answer questions like:

  • How many transactions occurred on each date?
  • How many products were purchased in each transaction?
  • Which products were bought together in the same transaction?

  • Transaction 53702 included 1 product (Potato)
  • Transaction 53703 included 2 products (Watermelon, Banana)

Data ko hierarchy ke hisaab se analyze karna bahut important hai. Pehle hum date se filter karte hain, phir transaction ID se sort karte hain. Is tarah se hum dekh sakte hain ki ek din mein kitne transactions hue, har transaction mein kitne products the, aur kaun se products ek saath kharide gaye. Bade datasets mein ye technique bahut kaam aati hai jab aap patterns dhoondhna chahte hain. Ye exploratory analysis ka ek zaroori hissa hai!

Summary and Applications

In this lecture, we learned several basic but essential data arrangement techniques:

  • Sorting: Arranging data in ascending or descending order
  • Filtering: Showing only data that meets specific criteria
  • Grouping: Organizing data at different hierarchical levels

These techniques are fundamental to exploratory data analysis and help you:

  • Get a better visual understanding of your dataset
  • Identify patterns and relationships between variables
  • Prepare data for more advanced analysis
  • Generate initial insights before applying statistical methods

While we used Excel for demonstrations, these concepts apply to other tools like Python, R, and various AI platforms you'll explore in lab sessions.

Toh dosto, aaj humne data ko arrange karne ke teen basic techniques seekhe - sorting, filtering, aur grouping. Ye techniques data analysis ki neev hain. Chahe aap Excel use karen, Python use karen, ya R use karen - ye concepts har jagah apply hote hain. In techniques ki madad se aap apne data ko behtar tarike se samajh sakte hain, patterns dhundh sakte hain, aur advanced analysis ke liye taiyar kar sakte hain. Agle lecture mein hum descriptive statistics ke baare mein aur details se baat karenge. Tab tak ke liye, ye concepts practice karte rahiye!
